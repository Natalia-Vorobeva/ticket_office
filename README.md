## 📌 О проекте

Ticket Office – это одностраничное приложение для выбора фильмов, дат сеансов, мест в зале и бронирования билетов. Все данные сохраняются в `localStorage`, поэтому после перезагрузки страницы выбор пользователя не теряется (до очистки кэша). Проект разработан в качестве pet‑проекта для демонстрации навыков frontend-разработки.

### Основные возможности

- 🎬 **Каталог фильмов** – список доступных фильмов с возможностью выбора и просмотра подробной информации в модальном окне.
- 📅 **Интерактивный календарь** – удобный выбор даты и сеанса с отображением занятых мест. Даты разбиты на мини-кнопки с указанием месяца и числа.
- 🪑 **Схема зала** – визуальный выбор мест с подсветкой свободных, занятых и VIP-мест. Занятые места загружаются из `localStorage` и отображаются сразу.
- ⏱️ **Таймер бронирования** – после выбора мест даётся 10 минут на подтверждение, иначе бронь сбрасывается.
- 💬 **Информационное окно** – после нажатия «Подтвердить» показывается сводка выбранного билета, после закрытия страница плавно прокручивается вверх.
- 📱 **Адаптивный дизайн** – комфортная работа на любых устройствах (десктоп, планшет, мобильный).
- 🧭 **Навигационное меню** – фиксированное меню для быстрого перехода к разделам «Фильмы», «Дата», «Зал», «Бронь».

## 🛠 Технологии

- **React 19** – функциональные компоненты, хуки (useState, useEffect, useMemo, useCallback, useRef)
- **Vite** – быстрая сборка и горячая перезагрузка
- **Tailwind CSS** – утилитарная стилизация без лишнего CSS
- **localStorage** – хранение данных о датах и занятых местах
- **Vercel** – хостинг для демо-версии

## 🚀 Запуск локально

### Предварительные требования
- Node.js (версия 18+)
- npm или yarn

### Клонирование репозитория
```bash
git clone https://github.com/Natalia-Vorobeva/ticket_office.git
cd ticket_office
```

### Установка зависимостей
```bash
npm install
# или
yarn
```
### Запуск в режиме разработки
```bash
npm run dev
```
