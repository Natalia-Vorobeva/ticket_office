# Билетная касса 
### В приложении можно выбрать фильм, день, сеанс, зал и забронировать места в выбранном зале.
### Данные хранятся в localStorage: текущая дата, предыдущие 7 дней и будущие 7 дней. То что вы вчера зарезервировали не потеряется, так как на следующий день обновится текущая дата, удалится 8 день из старых дат и появится новая дата в будущих.
### По предыдущим датам можно посмотреть заполненность зала.
